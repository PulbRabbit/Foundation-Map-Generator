WHAT IS IT:
It is a GUI Editor for the Game Foundation. it is released under the name UFME(Unofficial Foundation Map Editor)
You can create custom maps for the game and get all necessary files , without editing mod files directly.
It is also possible to import heightmaps and generate all necessary resource maps to it.

WHATS THE STATUS:
its released in version 1.2.1

WHAT ARE THE FILES:

assets - is a library for generating all those asset maps from the heightmap
terrain - is a library for all types of terrain for generation
map_gen - is basically a test file to test and watch new functions
gen_heightmap - generates a heightmap from the TerrainObjectList.jsonc file - deprecated
gen_assetmaps - generates assetmaps from the TerrainObjectList.jsonc file - deprecated
mapgen_guilib - is a library for the mayor gui elemets
graphic interface - the new main file, that provides the GUI Editor

The Editor is released on mod.io/foundation
a getting started can be found there
